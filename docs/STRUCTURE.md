# Project Structure

```
budget-app/
├── index.html              # Main HTML file
├── budget.js               # Core logic + DOM handlers
├── chart.js                # Chart visualization
├── style.css               # Styles
├── package.json            # Dependencies and scripts
├── __tests__/
│   └── budget.test.js      # Test suite
├── docs/                   # Documentation
├── .gitignore              # Git ignore rules
└── icon/                   # Assets
```

## Key Files

- `budget.js`: Main application logic
- `chart.js`: Circular chart rendering
- `style.css`: All CSS styles
- `index.html`: Entry point
